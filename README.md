# Pomodoro timer

## Getting started

1.  **Clone this repo and install the dependencies:**

    ```sh
    $ git clone https://github.com/hoflish/pomodoro-react
    $ cd pomodoro-react
    $ npm install

    ```

1.  **Start developing.**

    ```sh
    gatsby develop
    ```
    Your site is now running at `http://localhost:8000`!
